### Tastaturbelegung

Die Tasten sind wie folgt belegt. 

| Taster | Funktion    |
|--------|-------------|
| Up     | Temp ++     |
| Down   | Temp --     |
| Right  | Rast ++     |
| Left   | Rast --     |
| SELECT | Rast on/off |
| RST    | Reset       |
